pytest测试框架

发表于 2019-08-01 | 分类于 从零开始对一个web项目进行自动化接口测试系列 ， pytest测试框架

0、 安装

0.0、 pip install -U pytest

0.1、编写pytest测试用例，只需要按照下面的规则：

  • 测试文件以test_开头（以_test结尾也可以）
  • 测试类以Test开头，并且不能带有 init 方法
  • 测试函数以test_开头
  • 断言使用基本的assert即可

7、Console参数介绍

  • -v 用于显示每个测试函数的执行结果
  • -q 只显示整体测试结果
  • -s 用于显示测试函数中print()函数输出
